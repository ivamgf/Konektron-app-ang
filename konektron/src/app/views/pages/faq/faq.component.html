    Principais dúvidas em relação ao Konektron.

    O que é o Konektron?
    Konektron é um template de desenvolvimento rápido para e-commerce que servirá à diversos projetos dos mais variados segmentos.

    Como usar o Konektron?
    Para saber como usar o Konektron leia os termos de uso que poderá ser encontrado neste site.

    Quais as dependências tecnológicas do Konektron?
    Para desenvolver com o Konektron, utilize, de preferência as seguintes ferramentas de mercado e padrões Web Standart, como HTML5, CSS3, Sass, JavaScript, JSON, Angular 7+, Bootstrap versão 4+, PHP 7+, NodeJS, Wordpress (Blog), MySql, DBVisualizer, Laravel, Composer, Protractor, Npm, Karma, Jasmine e Visual Studio Code.

    Com quem devo falar se houver algum problema?

    Como faço para contribuir com o projeto Konektron?

    Como faço para contribuir financeiramente com o projeto Konektron?
